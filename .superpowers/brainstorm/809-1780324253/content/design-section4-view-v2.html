互動行為設計

Hover（滑鼠移入）
顯示 EquipmentTooltip — 快速摘要
內容：設備名稱、目前狀態、最近一筆警報
Click（點擊設備圖示）
展開 EquipmentDetailDrawer — 側欄詳細資訊
可設定多個 Link 頁籤（工單、警報原因、自訂連結...）

新增模型：EquipmentLinkConfig

EquipmentLinkConfig — 設備資訊連結設定
Id, EquipmentId
LinkType(Enum): WorkOrder, AlarmHistory, Document, CustomUrl
TabLabel　← 頁籤顯示名稱（如「工單」）
UrlTemplate　← 連結 URL 樣板（可含 {equipmentId} 參數）
DataSourceConfigId（nullable）← 可用資料源查詢顯示清單
DisplayOrder

LinkType Enum 新增：
WorkOrder, AlarmHistory, Document, CustomUrl

新增元件

EquipmentTooltip.razor　← Hover 快覽
設備名稱、狀態、最近警報摘要
mouseover / mouseout 觸發顯示/隱藏
EquipmentDetailDrawer.razor　← Click 詳細面板
側欄 Drawer，依 EquipmentLinkConfig 動態產生頁籤
每個頁籤可顯示：清單資料（工單）、文字說明（警報原因）、超連結
關閉按鈕或點擊地圖空白處收起

新增介面方法

IEquipmentService 新增：
Task<IEnumerable<EquipmentLinkConfig>> GetLinkConfigsAsync(equipmentId)
Task SaveLinkConfigAsync(EquipmentLinkConfig config)
Task DeleteLinkConfigAsync(id)
