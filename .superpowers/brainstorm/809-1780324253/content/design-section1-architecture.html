模組註冊流程

TPS.Nexus 模組啟動流程
1. TPS.Nexus.Web.exe 啟動
2. 掃描並載入 TPS.Nexus.Kanban.Web.dll
3. 執行 KanbanModuleRegistrar : IModuleRegistrar
→ 註冊 Blazor 元件路由
→ 註冊 Services DI（IMapImportService、ILayoutService 等）
→ 掛載 SignalR Hub（/hubs/kanban-alarm）
4. 使用者瀏覽 /kanban → 顯示看板地圖頁面

資料流概覽

執行期資料流
一般數值（輪詢）：
Timer 觸發 → DataSourceService.FetchAsync()
→ SQL / CSV / JSON / XML 適配器
→ 更新 Blazor 元件狀態 → 畫面重繪

警報（即時推送）：
資料源變動 → AlarmDetectionService
→ KanbanAlarmHub.SendAlarmAsync()
→ SignalR 推送 → 所有連線用戶端即時更新

佈局版本管理：
編輯模式拖拉 → LayoutService.SaveDraftAsync()
→ 審核後 LayoutService.PublishAsync()
→ 建立新版本，舊版本保留可回溯
